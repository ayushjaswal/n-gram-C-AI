# N-Gram Language Model

A simple trigram text generator for Mission 5: The Babel-Scale Crisis (Conversational AI WiSe 25/26).

## Files

- `ngram.py` - Trigram model implementation with Laplace smoothing
- `transformer.txt` - Training corpus (Wikipedia text on transformers)
- `response.txt` - Generated text samples

## Usage

```bash
python ngram.py
```

Generates text by predicting the next word based on the previous two words using probability distributions learned from the training data.
